# Utilitario - Cuentas por Cobrar y por Pagar

Es una aplicación de escritorio desarrollada para generar reportes detallados y específicos a partir de bases de datos del sistema contable **Versat Sarasola**. Diseñada con una interfaz moderna y amigable, este sistema permite a usuarios contables obtener informes útiles para la toma de decisiones financieras.

## Características

- Interfaz amigable y moderna
- Generación de múltiples tipos de reportes:
  - GER
  - MINDUS
  - Fuera de Organismo
  - Cooperativas
  - Personas Naturales
  - Cuentas Vencidas
  - Informes 102 y 203
- Sistema de licencias (requiere contacto con distribuidor)
- Basado en bases de datos del sistema contable **Versat Sarasola**
- Utiliza SQL Server 2017 o 2019
- Rápido, eficiente y portable

## ¿Cómo usar?

```bash
git clone git@github.com:A4GOD-AMHG/Utilitario-CxC-CxP.git

cd Utilitario-CxC-CxP/frontend

npm install

cd ..

wails dev
```

- Compila el proyecto usando wails dev o wails build
- Ejecuta el binario generado o ejecuta en modo desarrollo
- (Opcional) Solicita licencia al distribuidor para utilizar la versión completa
